# OpenAgents

An open platform for AI agents, built in public from scratch.

See the [wiki](https://github.com/ArcadeLabsInc/openagents/wiki) for more.

## Tech Stack
- Laravel
- React
- Inertia
- Tailwind CSS
- shad-ui
- PostgreSQL
